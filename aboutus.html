About Us

Project Overview

This web project is designed to help teachers manage students and analyze class performance. The dashboard provides a seamless way to view student information, edit student data, and monitor class analytics in various subjects like Math, Physics, Chemistry, Biology, and English.

The "Manage Students" section allows teachers to add, edit, or delete student records. Each student has key information like student ID, name, and department. Teachers can easily interact with the data through buttons for each action. The "Class Performance Analytics" section presents data visualizations, such as charts and lollipop graphs, showing how well the class is performing in each subject.

What This Project Provides

  • Manage Students: Allows the teacher to manage student details including ID, name, and department.
  • Class Analytics: Provides performance data across various subjects with interactive charts.
  • Interactive Interface: Easy-to-use interface to manage and visualize data effectively.
  • Secure Login: Ensures only authorized users (teachers) have access to the dashboard.
